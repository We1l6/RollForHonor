Detailed Description

UI skin class for visual styling of interface elements.

Encapsulates visual properties including color, texture, and segment detail used for rendering UI components. Supports initialization with either a solid color or a texture. Provides getters and setters to access and modify skin properties such as color, texture, and segment amount. The Draw() method renders the skin onto a specified RectanglePro area using raylib and rlgl.

Note
setSegmentAmount() clamps values to a minimum of 3. Use clearTexture() to disable texture usage.
